# ProyectoRLeon

Monorepo oficial del proyecto HRCloud.

## Estructura
- /cerebro-front   -> panel SaaS proveedor
- /base-front      -> app cliente Base
- /adms-gateway    -> gateway biométrico / ADMS
- /pwa             -> app móvil / PWA
- /supabase        -> SQL, config y edge functions

## Notas
- No subir secretos ni archivos .env reales
- DigitalOcean App Platform debe apuntar por Source Directory
